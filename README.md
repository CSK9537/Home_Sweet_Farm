# 🌿 Home Sweet Farm (홈 스윗 팜)

## 📝 프로젝트 개요

**Home Sweet Farm**은 1인 가구의 증가에 따른 반려식물 문화 확산에 발맞춰, 초보자도 실패 없이 식물을 키울 수 있도록 돕는 **IoT 기반 반려식물 올인원 관리 서비스**입니다. 

단순히 감에 의존하는 관리가 아닌, 아두이노 센서 데이터를 활용한 실시간 모니터링과 식물 종류별 맞춤형 정보를 제공하여 식물 폐사율을 낮추고, 전문가 상담 및 커뮤니티를 통해 건강하고 책임감 있는 반려식물 문화를 조성하는 것을 목표로 합니다.

## 💻 기술 스택

### Back-end
* Java 11
* Spring Framework (Spring MVC, Spring Security)
* MyBatis
* WebSocket
* Jackson

### Front-end
* JSP, JSTL, EL
* HTML / CSS / JavaScript

### Database
* Oracle DB

### Hardware / IoT
* Arduino

## ⚙️ 주요 기능

* **실시간 식물 관리 시스템 (IoT)**: 아두이노 센서를 통해 토양 수분, 온도 등을 실시간으로 측정하고 직관적인 대시보드로 시각화하여 제공합니다.
* **식물 백과사전 및 가이드**: 각 식물의 특성에 따른 맞춤형 관리 가이드와 방대한 백과사전 정보를 제공합니다.
* **1:1 실시간 채팅**: 웹소켓(WebSocket) 기반의 실시간 채팅 기능을 통해 즉각적으로 식물 상태를 상담할 수 있습니다.
* **커뮤니티 및 전문가 Q&A**: 인기글/최신글 기반의 정보 공유 게시판과 전문가에게 질문을 통해 식물 상태에 대한 답변을 받을 수 있습니다.
* **스토어 서비스**: 현재 내 식물 상태에 꼭 필요한 영양제 및 관리 용품을 추천받고 바로 구매할 수 있습니다.
